Deep Explanation

Unlike RNNs that process data step-by-step (linearly), Transformers employ an attention mechanism that allows them to weigh the significance of each part of the input data regardless of its position in the sequence. This "parallelization" is what makes them significantly faster to train and more effective at capturing long-range dependencies in text.
